CryptoJournal 
==============
CryptoJournal is an all in one cryptocurrency and coin news directory. 

Specifications
--------------

- Pure Proof of Stake
- X11 algorithm
- 100.000.000 CJC premine sold at YoBit ICO
- 60 sec blocktime
- 14 CJC fixed Proof of Stake block reward
- 6 hours stake min. age
- 10 days stake max. age
- RPC Port - 6724
- Default Port - 6725
